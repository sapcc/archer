Request Body schema: application/json

Service object that needs to be added to the catalog

description
string

Description of the service.

ports
Array of integers

Ports exposed by the service.

network_id
string <uuid>

Network ID of the network that provides this service.

ip_address
string <ipv4>

IP Address of the providing service.

string = 32 characters

The ID of the project owning this resource.

Inject a service into a target network.

Authorizations:
api_key
Request Body schema: application/json

Service and target network to inject. Only one of target_network, target_subnet or target_port must be specified.

service_id
string <uuid>

The ID of the service.

target_network
string <uuid>

Injection network target. One of target_network, target_subnet or target_port must be specified.

target_subnet
string <uuid>

Injection subnet target. One of target_network, target_subnet or target_port must be specified.

target_port
string <uuid>

Injection port target. One of target_network, target_subnet or target_port must be specified.

project_id
string = 32 characters

The ID of the project owning this resource.
